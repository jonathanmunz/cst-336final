NAT stands for Networking Address Translation

NAT is primarily used for converting private ip addresses to public ip addresses.

So why do we need NAT?.

When the internet was first being developed, the first widespread IP addressing system used was known as IPv4. This allowed for the creation of over 4 billion IP addresses. The creators of IPv4 assumed that this would be enough IP addresses to last forever, they were wrong. The internet grew and IP addresses began to run out! So what was the solution? Well ultimately it has become IPv6 which allows for enough IP addresses for every grain of sand on the earth to have one! But before that along came NAT. NAT allows for differant end devices (Computers) on differant networks to use the same IP address. How? with the use of private and public IP addresses. Instead of forcing everyone to have a unique IP address, they only need to have a unique public IP address. They can use the same private IP address as long as they are on a differant network. This extended the pool of IPv4 addresses for many years.
